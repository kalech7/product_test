# Products 

Aplicación web para **gestión de productos** (listar, crear, editar, eliminar) construida con **React 18**, **TypeScript** y el ecosistema **TanStack**.  
Usa una **API simulada en TypeScript** 

---

## Tecnologías

- React 18 + TypeScript
- Vite
- TanStack Router
- TanStack Query
- React Hook Form + Zod
- Zustand
- Tailwind CSS
- Axios
- React Toastify

---

```javascript
npm install
npm run dev
